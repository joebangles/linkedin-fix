# LinkedIn Fix

Trying to play [Zip](https://www.linkedin.com/games/zip) in the middle of the night and being bombarded with the success of others? Pretend your problems don't exist.

## Features

- Hides the main feed on LinkedIn homepage

## Installation

1. Clone or download this repository
2. Open `chrome://extensions` in Chrome
3. Enable **Developer mode** (top-right toggle)
4. Click **Load unpacked** and select the project folder

## Usage

Click the extension icon to toggle it on or off. Changes take effect on the next page load.
